📊 INTRO

Tu negocio en una pantalla

Un dashboard es una hoja de resumen que muestra lo más importante de un vistazo: ventas del mes, meta alcanzada, productos más vendidos, días con más ingresos. Sin abrir 20 hojas, sin hacer cálculos manuales.

La idea: Tienes una hoja "Datos" donde registras ventas. Una hoja "Dashboard" que las lee con fórmulas y las muestra bonito. Cada vez que agregas una venta en "Datos", el dashboard se actualiza solo.

🔢 FÓRMULAS KPI

Construye tus KPIs con fórmulas

En tu hoja "Dashboard", estas fórmulas leen los datos y calculan lo que necesitas ver. Todas apuntan a tu hoja "Datos".

Fórmulas para tu Dashboard
// Ventas totales del mes
=SUMAR.SI.CONJUNTO(Datos!C:C,Datos!A:A,">="&DATE(2025,6,1),Datos!A:A,"<="&DATE(2025,6,30))
 
// % de meta alcanzada (meta en celda B2)
=TEXTO(B5/B2,"0%")
 
// Pedidos pendientes
=CONTAR.SI(Datos!D:D,"Pendiente")
 
// Producto más vendido
=INDICE(Datos!B:B,COINCIDIR(MAX(CONTAR.SI(Datos!B:B,Datos!B:B)),CONTAR.SI(Datos!B:B,Datos!B:B),0))
 
// Promedio diario de ventas
=PROMEDIO.SI(Datos!A:A,">="&DATE(2025,6,1),Datos!C:C)

🎨 FORMATO CONDICIONAL

Que se pinte solo según el resultado

El formato condicional colorea celdas automáticamente: verde si vas bien, amarillo si debes acelerar, rojo si hay problema. Sin hacer nada manualmente.

1

Selecciona las celdas a colorear

Ej: la celda con el % de meta alcanzada (B6).

2

Formato → Formato condicional

Aparece el panel de reglas a la derecha.

3

Agrega las 3 reglas

Una por color: rojo si <50%, amarillo si entre 50-80%, verde si >80%.

Condición Valor Color Significado
Es menor que 0.5 (50%) 🔴 Rojo Meta en riesgo
Entre 0.5 y 0.8 🟡 Amarillo Hay que acelerar
Es mayor o igual que 0.8 (80%) 🟢 Verde Va bien
✨

Aplícalo también a filas completas: En "Datos", colorea toda la fila de rojo si el estado es "Atrasado". Selecciona el rango → Formato condicional → Fórmula personalizada → =$D1="Atrasado"

📈 GRÁFICAS

Gráficas que se actualizan solas

Las gráficas de Sheets se conectan a tus datos. Cuando agregas una venta, la gráfica se actualiza sola — sin tocar nada.

1

Crea una tabla resumen por semana

En el Dashboard: Semana 1, 2, 3, 4 en col A. En col B, usa SUMAR.SI.CONJUNTO para cada período.

2

Selecciona las dos columnas → Insertar → Gráfica

Sheets propone la mejor gráfica automáticamente. Elige barras para ventas por período.

3

Personaliza colores y etiquetas

Doble clic en la gráfica → Editor de gráficas. Cambia colores, agrega título, muestra valores.

4

Mueve la gráfica al Dashboard

La gráfica puede vivir en cualquier hoja. Arrástrala al Dashboard para tenerla con tus KPIs.

💡

Truco: Usa QUERY para agrupar datos por mes o semana automáticamente. Pregúntale a Claude: "¿Cómo agrupo ventas por semana en Google Sheets para una gráfica?"

🚨 ALERTAS

Te avisa cuando algo va mal

El dashboard muestra el problema. Las alertas te avisan aunque no estés mirando la pantalla. Este script revisa tus KPIs y manda un correo si algo está por debajo del umbral.

Apps Script — Sistema de alertas del dashboard
function verificarAlertas() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const dash = ss.getSheetByName("Dashboard");
  
  // Lee los KPIs del Dashboard (ajusta celdas según tu hoja)
  const ventasMes = dash.getRange("B5").getValue();
  const meta = dash.getRange("B2").getValue();
  const pendientes = dash.getRange("B8").getValue();
  
  const porcentajeMeta = ventasMes / meta;
  let alertas = [];
  
  if (porcentajeMeta < 0.5) {
    alertas.push(`🔴 CRÍTICO: Solo ${(porcentajeMeta*100).toFixed(0)}% de la meta alcanzada`);
  } else if (porcentajeMeta < 0.8) {
    alertas.push(`🟡 ATENCIÓN: ${(porcentajeMeta*100).toFixed(0)}% de la meta — hay que acelerar`);
  }
  
  if (pendientes > 20) {
    alertas.push(`⚠️ Tienes ${pendientes} pedidos pendientes sin atender`);
  }
  
  if (alertas.length > 0) {
    const cuerpo = `📊 ALERTA DE DASHBOARD\n\n`
      + alertas.join("\n") + "\n\n"
      + `Ventas mes: Q${ventasMes} / Meta: Q${meta}\n`
      + `Revisa tu dashboard: ${ss.getUrl()}`;
    
    GmailApp.sendEmail(
      "tu@email.com",
      "🚨 Alerta de Dashboard — Acción necesaria",
      cuerpo
    );
  }
}
⏰

Configura el disparador: Activa este script para que corra cada mañana a las 7am. Si todo está bien, no manda nada. Si algo está mal, llegas a trabajar con el correo de alerta esperándote.
