🧩 Sección 1

¿Qué son los MCPs y por qué te cambian la vida?

Por defecto, Claude vive en una "caja": sabe muchísimo, pero no puede conectarse con el exterior. No puede navegar la web, leer tus repositorios en tiempo real ni buscar noticias de hoy. Los MCPs rompen esa caja.

El Model Context Protocol (MCP) es un estándar abierto creado por Anthropic que permite a Claude conectarse con herramientas externas del mundo real: navegadores, bases de datos, APIs, servicios en la nube y mucho más. En Claude Code, los MCPs son especialmente poderosos porque convierten al asistente en un agente autónomo.

🌐

Navega la web

Con Playwright MCP, Claude puede abrir páginas, hacer clic, rellenar formularios y extraer información de cualquier sitio.

🔍

Busca en tiempo real

Con Perplexity MCP obtiene información actualizada con fuentes verificadas, no solo lo que aprendió antes de su entrenamiento.

🐙

Gestiona código

GitHub MCP le da control total sobre repositorios: issues, pull requests, commits y revisión de código automática.

🧠

Razona mejor

Sequential Thinking MCP hace que Claude articule su proceso mental paso a paso y pueda corregirse a sí mismo.

💡

Analogía simple: Claude sin MCPs es como un experto encerrado en una sala sin internet. Con MCPs, ese mismo experto tiene acceso a todo: buscador, GitHub, un navegador, y herramientas de razonamiento avanzado. El experto es el mismo — las herramientas lo hacen 10× más útil.

⚡ Sección 2

Cómo instalar un MCP en 1 minuto

Todos los MCPs de esta guía se instalan con un único comando. El único requisito previo es tener Node.js 18+ instalado en tu computadora.

1

Verifica que tienes Node.js

Abre tu terminal y escribe node --version. Si ves algo como v20.x.x, estás listo. Si no, descárgalo gratis en nodejs.org.

2

Ejecuta el comando de instalación

Cada MCP tiene su propio comando. Lo encontrarás en las secciones siguientes. El formato general es claude mcp add [nombre] [comando].

3

Reinicia Claude Code

Cierra y vuelve a abrir Claude Code. El MCP estará disponible automáticamente. Sin configuración adicional en la mayoría de los casos.

4

Verifica que funciona

Escríbele a Claude algo relacionado con el MCP instalado. Por ejemplo, con Playwright: "Abre google.com y dime el título de la página". Verás que ahora puede hacerlo.

🚀 Sección 3

Los 5 MCPs imprescindibles

Estos son los MCPs más útiles para quien trabaja con Claude Code a diario. Cada uno tiene su caso de uso específico — y juntos forman un arsenal completo.

🎭
MCP 01 · Microsoft Oficial

Playwright MCP — Automatización de navegador

Convierte a Claude Code en un agente de navegador completo. Opera sobre el árbol de accesibilidad del navegador (no capturas de pantalla), lo que lo hace preciso y compatible con cualquier modelo.

Úsalo para: scraping de webs dinámicas, testing automático, rellenar formularios, monitorear cambios en páginas, automatizar dashboards.

💻 Comando de instalación
claude mcp add playwright npx @playwright/mcp@latest
Sin API key Requiere Node.js 18+
🔥
MCP 02 · Firecrawl Oficial

Firecrawl MCP — Scraping y búsqueda avanzada

Convierte cualquier página web en datos limpios y estructurados (Markdown o JSON). Soporta JavaScript, paginación y extracción masiva. Ideal para investigación y análisis de competidores.

Úsalo para: analizar webs de competidores, crear datasets, monitorear precios, extraer documentación técnica, resumir artículos o blogs automáticamente.

💻 Comando de instalación
claude mcp add firecrawl npx firecrawl-mcp
# Necesitas una API key gratis en firecrawl.dev (500 créditos gratis)
API key gratis Requiere Node.js 18+
🔮
MCP 03 · Perplexity Oficial

Perplexity MCP — Búsqueda con razonamiento en tiempo real

Conecta Claude Code con el motor de Perplexity AI: información actualizada con citas verificadas. A diferencia de una búsqueda común, sintetiza respuestas combinando múltiples fuentes con razonamiento propio.

Úsalo para: investigación técnica con fuentes, noticias del sector IA, fact-checking, tendencias de mercado, documentación oficial y papers académicos.

💻 Comando de instalación
claude mcp add perplexity-search --env PERPLEXITY_API_KEY=tu_key -- npx -y server-perplexity-ask
API key de Perplexity Requiere Node.js 18+
🐙
MCP 04 · GitHub Oficial

GitHub MCP — Control total de repositorios

Interactúa directamente con repositorios, issues, pull requests y GitHub Actions. La herramienta definitiva para flujos de DevOps con IA: gestiona código, revisa PRs y automatiza tareas sin salir de Claude.

Úsalo para: code review automático, gestión de issues, documentación, changelogs, buscar patrones en repositorios open source.

💻 Comando de instalación
claude mcp add github npx @github/github-mcp-server
# Necesitas un Personal Access Token de GitHub Settings > Developer Settings
GitHub Token Requiere Node.js 18+
🧩
MCP 05 · Anthropic Oficial

Sequential Thinking MCP — Razonamiento estructurado

Implementa un protocolo de razonamiento dinámico: Claude articula su proceso mental en pasos explícitos, puede revisar conclusiones anteriores y explorar hipótesis alternativas. Ideal para problemas complejos.

Úsalo para: debugging complejo paso a paso, arquitecturas de software, problemas matemáticos, análisis de requisitos, decisiones técnicas con pros/contras explícitos.

💻 Comando de instalación
claude mcp add sequential-thinking npx @modelcontextprotocol/server-sequential-thinking
Sin API key Requiere Node.js 18+
📊 Sección 4

Comparativa rápida: ¿cuál instalar primero?

Si estás empezando, no instales los 5 a la vez. Empieza con los que no requieren API key — son instantáneos — y añade los demás según tu caso de uso.

MCP Categoría API Key Dificultad Para empezar
🎭 Playwright Automatización web No necesita Media ✓ Sí
🔥 Firecrawl Scraping avanzado Gratis en web Fácil ✓ Sí
🔮 Perplexity Búsqueda IA Pago (pplx.ai) Fácil Segundo paso
🐙 GitHub DevOps & Código Token GitHub Media Si usas GitHub
🧩 Seq. Thinking Razonamiento No necesita Muy fácil ✓ Primero
🏁

Recomendación Jonathan: Empieza con Sequential Thinking (sin API key, instalación instantánea) y nota de inmediato cómo Claude razona mejor. Luego añade Playwright para automatizar la web. Una vez con eso dominado, los otros tres te abrirán casos de uso completamente nuevos.
