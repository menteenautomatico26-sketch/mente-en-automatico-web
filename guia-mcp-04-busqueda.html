❓ Sección 1

El problema: Claude vive en el pasado

Claude es increíblemente capaz, pero tiene dos limitaciones conocidas: su conocimiento tiene fecha de corte (no sabe qué pasó esta semana), y en problemas muy complejos a veces "salta" a conclusiones sin razonar bien. Estos dos MCPs resuelven exactamente eso.

Sin estos MCPs

"¿Cuál es el precio actual de [producto]?" → Claude inventa o dice que no sabe

"Analiza este problema complejo" → Respuesta directa sin mostrar el razonamiento

Con Perplexity + Sequential Thinking

"¿Cuál es el precio actual de [producto]?" → Busca en tiempo real, da el precio con fuente

"Analiza este problema complejo" → Razona paso a paso, muestra su proceso, puede corregirse

🔮 Sección 2

Perplexity MCP — Búsqueda con razonamiento en tiempo real

Perplexity no es solo un buscador — es un motor que sintetiza múltiples fuentes con razonamiento propio. Cuando se lo das a Claude, obtiene información actualizada con citas verificadas en lugar de respuestas genéricas de su entrenamiento.

⚡

Tiempo real

Noticias de hoy, precios actuales, lanzamientos recientes. No más respuestas desactualizadas.

📚

Fuentes verificadas

Cada respuesta incluye las fuentes. Puedes verificar la información directamente.

🔬

Papers académicos

Accede a documentación técnica, papers y artículos científicos recientes.

📊

Sonar Pro

El modelo sonar-pro de Perplexity da respuestas más completas y con mayor contexto para temas complejos.

Instalación de Perplexity MCP

1

Obtén tu API key de Perplexity

Ve a pplx.ai → crea una cuenta → en el panel de control encuentra la sección API → genera tu API key. El plan de pago comienza desde $5/mes con buen volumen de consultas.

2

Instala el MCP con tu API key

💻 Terminal
claude mcp add perplexity-search \
  --env PERPLEXITY_API_KEY=tu_key_aqui \
  -- npx -y server-perplexity-ask
3

Reinicia Claude Code y prueba

Escríbele: "¿Qué novedades importantes salieron sobre IA esta semana?" Si te responde con eventos recientes con fuentes, está funcionando.

Prompts recomendados con Perplexity activo

📰 Noticias del sector
"Busca las 5 noticias más importantes sobre IA generativa de los últimos 7 días. Dame un resumen de cada una con la fuente."
🏆 Análisis de competidores
"Busca información actualizada sobre los precios y características de [competidor]. Necesito una comparativa con lo que ofrece [mi producto]."
📖 Investigación técnica
"Encuentra los papers más citados sobre [tema técnico] publicados en 2025. Dame los títulos, autores, dónde están publicados y el hallazgo principal de cada uno."
🧩 Sección 3

Sequential Thinking MCP — Razona paso a paso

Sequential Thinking implementa un protocolo de razonamiento dinámico: Claude articula su proceso mental en pasos numerados y revisables, puede volver atrás y corregir conclusiones incorrectas, y explora hipótesis alternativas. Es el MCP más subestimado de todos — y el que más impacto tiene en la calidad de las respuestas.

🧠

¿Cuándo marca la diferencia? En problemas con múltiples pasos, decisiones técnicas con trade-offs, debugging complejo, planificación de proyectos, y cualquier análisis donde el "por qué" importa tanto como el "qué".

Instalación (sin API key — completamente gratis)

💻 Terminal — 1 solo comando
claude mcp add sequential-thinking npx @modelcontextprotocol/server-sequential-thinking
✅

Sin API key. Sin pago. Sin configuración adicional. Reinstala Claude Code y estará disponible. Es el MCP con mejor relación instalación/impacto de todos los que existen.

Cómo activarlo en tus prompts

🔍 Debugging paso a paso
"Piensa paso a paso: este código produce el error [X]. Analiza cada función involucrada, identifica dónde falla y por qué antes de proponer la solución."
🏗️ Decisión de arquitectura
"Analiza en detalle los pros y contras de usar [opción A] vs [opción B] para [mi caso de uso]. Considera escalabilidad, costo, mantenimiento y curva de aprendizaje del equipo."
📋 Planificación de proyecto
"Desglosa paso a paso cómo construiría [funcionalidad X]. Identifica dependencias, posibles bloqueos y el orden óptimo de implementación."
🚀 Sección 4

El combo definitivo: los dos juntos

La combinación más poderosa: Perplexity da información actualizada y con fuentes, mientras Sequential Thinking hace que Claude procese esa información de forma rigurosa y estructurada antes de darte la respuesta final.

1

Investigación técnica profunda

Pídele que busque información reciente sobre un tema técnico (Perplexity) y luego que la analice paso a paso para darte una recomendación fundamentada (Sequential Thinking).

2

Análisis de tendencias de mercado

Busca los últimos movimientos de tu sector con Perplexity y pídele que razone estratégicamente sobre las implicaciones para tu negocio, paso a paso.

3

Fact-checking riguroso

Verifica afirmaciones o datos con Perplexity y pídele que evalúe la calidad y credibilidad de las fuentes usando razonamiento explícito.

⚡ Prompt combo — Perplexity + Sequential Thinking
"Busca información actualizada sobre el estado actual de [tema X] en 2025.
Luego, piensa paso a paso qué implicaciones tiene esto para alguien que trabaja en [mi contexto].
Cita las fuentes de la información que uses y muestra tu razonamiento explícitamente."
