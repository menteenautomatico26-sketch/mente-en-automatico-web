# Mente en Automático — Sitio web

Sitio estático (HTML/CSS/JS) de **Mente en Automático · IA & Eficiencia**.
Tienda de plantillas, novedades de IA y contacto.

## Estructura
- `index.html` — inicio (sobre mí, productos, recursos, contacto)
- `tienda.html` — tienda con plantillas y packs
- `novedades.html` — blog de novedades y tips de IA
- `gracias.html` — página post-compra
- `styles.css` · `app.js` — estilos y lógica (catálogo, blog, partículas)

## Despliegue
Conectado a Netlify. Cada push a la rama principal actualiza el sitio en vivo.

## Configuración
Edita `app.js` → `CONFIG` (WhatsApp, Instagram, correo y links de pago).
