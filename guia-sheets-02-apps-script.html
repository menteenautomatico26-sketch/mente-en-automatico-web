⚙️ Sección 1

¿Qué es Apps Script y para qué sirve?

Google Apps Script es el motor de automatización de Google. Es código JavaScript que vive dentro de tus Sheets, Docs o Gmail — y puede hacer cosas que las fórmulas no pueden.

📧

Enviar correos automáticos

Manda reportes diarios, alertas de stock bajo o confirmaciones de pedidos — sin tocar nada.

⏰

Ejecutarse solo

Configura que el script corra todos los días a las 8am, o cada vez que alguien edita la hoja.

📋

Mover datos entre hojas

Copia registros de una hoja a otra según condiciones. Archiva automáticamente filas completadas.

🔔

Crear alertas inteligentes

Si el inventario baja de cierto nivel, te manda un WhatsApp (vía API) o un correo instantáneo.

🤖

La clave: No necesitas aprender a programar. Con Claude, describes en español lo que quieres que haga tu Sheet, él te escribe el código exacto, y tú lo pegas en el editor. Así de simple.

🚀 Sección 2

Cómo abrir el editor en 30 segundos

1

Abre tu Google Sheet

Cualquier hoja de cálculo que tengas en Google Drive. Si no tienes ninguna, crea una nueva en sheets.google.com.

2

Extensiones → Apps Script

En el menú superior: Extensiones → Apps Script. Se abre una nueva pestaña con el editor de código.

3

Verás esto (normal)

function myFunction() {
  // Tu código va aquí
}

Esto es el punto de partida. Borra todo y pega el código que te dé Claude.

4

Clic en ▶ Ejecutar

Después de pegar el código de Claude: clic en el botón de play (▶) o presiona Ctrl+R. La primera vez te pedirá permisos — acéptalos, son para que el script pueda acceder a tu Sheet.

🤖 Sección 3

El flujo: Claude escribe, tú pegas

Este es el proceso exacto para cualquier automatización que quieras crear:

1

Describe lo que quieres en español

No uses términos técnicos. Explícalo como se lo explicarías a un asistente: "Quiero que cuando la columna C diga Completado, copie esa fila a la hoja Archivo."

2

Dale contexto de tu Sheet a Claude

Dile los nombres de tus columnas y hojas. Ejemplo: "Mi Sheet tiene columnas: A=Fecha, B=Cliente, C=Estado, D=Monto. Tengo dos hojas: Pedidos y Archivo."

3

Pega el código en el editor

Claude te dará el código completo. Cópialo todo, abre el editor de Apps Script (Extensiones → Apps Script), borra lo que hay y pega el nuevo código.

4

Prueba y ajusta

Ejecuta el script. Si hay algún error, cópialo y mándalo de vuelta a Claude: "Me da este error: [error]". Él lo corrige.

⚡ Sección 4

3 automatizaciones que puedes activar hoy

Estas son las más útiles para empezar. Cada una incluye el código listo para pegar.

Automatización 01

📧 Resumen diario por correo

Cada mañana a las 8am te manda un correo con el resumen del día anterior: total de pedidos, monto total, cuántos pendientes.

⚙️ Código Apps Script
function enviarResumenDiario() {
  const hoja = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Pedidos");
  const datos = hoja.getDataRange().getValues();
  let total = 0, pendientes = 0;
  
  for (let i = 1; i < datos.length; i++) {
    total += datos[i][3]; // Columna D = Monto
    if (datos[i][2] === "Pendiente") pendientes++;
  }
  
  const mensaje = `Resumen del día:\n- Total pedidos: ${datos.length-1}\n- Monto total: Q${total}\n- Pendientes: ${pendientes}`;
  GmailApp.sendEmail("tu@email.com", "📊 Resumen diario", mensaje);
}

💡 Para que corra solo: en Apps Script → Activadores (reloj) → Agregar activador → Basado en tiempo → Diario → 8am

Automatización 02

⚠️ Alerta de stock bajo

Revisa tu inventario y te manda correo cuando algún producto baja del mínimo que defines.

⚙️ Código Apps Script
function alertaStockBajo() {
  const hoja = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Inventario");
  const datos = hoja.getDataRange().getValues();
  const MINIMO = 10; // Cambia este número según tu necesidad
  let alertas = [];
  
  for (let i = 1; i < datos.length; i++) {
    if (datos[i][2] < MINIMO) { // Columna C = Stock
      alertas.push(`⚠️ ${datos[i][0]}: solo ${datos[i][2]} unidades`);
    }
  }
  
  if (alertas.length > 0) {
    GmailApp.sendEmail("tu@email.com", "🔴 Stock bajo", alertas.join("\n"));
  }
}
Automatización 03

📁 Archivar filas completadas

Mueve automáticamente a otra hoja las filas donde el Estado es "Completado". Tu hoja principal siempre queda limpia.

⚙️ Código Apps Script
function archivarCompletados() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const origen = ss.getSheetByName("Pedidos");
  const destino = ss.getSheetByName("Archivo");
  const datos = origen.getDataRange().getValues();
  
  for (let i = datos.length - 1; i >= 1; i--) {
    if (datos[i][2] === "Completado") { // Columna C = Estado
      destino.appendRow(datos[i]);
      origen.deleteRow(i + 1);
    }
  }
}
