📧 INTRO

El correo manual es tiempo muerto

Cada vez que confirmas un pedido, avisas que un pago venció o mandas el resumen del mes — estás repitiendo exactamente lo mismo. Apps Script + Gmail te permite hacer eso una sola vez y olvidarte para siempre.

🛒

💰

Cobros y finanzas

Avisa automáticamente cuando una factura lleva 3, 7 o 15 días sin pagar.

📊

Reportes de equipo

Manda el resumen de ventas cada lunes a las 8am sin que nadie lo active.

📅

Reservaciones

Confirma citas de clientes con todos los detalles en segundos.

💡

La clave: Google Sheets ya tiene los datos. Apps Script solo los toma, los mete en una plantilla de correo y usa Gmail para enviarlo. Claude te escribe el código completo — tú solo lo copias y pegas.

🛒 SCRIPT 1

Confirmación de pedido automática

Tu Sheet tiene: Nombre del cliente (col A), Email (col B), Producto (col C), Total (col D), Estado (col E). Cuando el estado cambia a "Confirmado", sale el correo.

Prompt para Claude
Tengo un Google Sheet con estas columnas: A: Nombre, B: Email, C: Producto, D: Total (Q), E: Estado Crea un script de Apps Script que: 1. Recorra todas las filas desde la 2 2. Si la columna E dice "Confirmado" y la columna F está vacía 3. Envíe un correo al email de col B con: - Asunto: "✅ Pedido confirmado - [nombre]" - Cuerpo con el nombre, producto y total - Mensaje de que estará listo en 3-5 días hábiles 4. Marque col F con "Enviado" para no mandar el correo dos veces Haz el código completo listo para pegar en Apps Script.
Apps Script — Confirmación de pedido
function enviarConfirmaciones() {
  const hoja = SpreadsheetApp.getActiveSpreadsheet()
    .getSheetByName("Pedidos");
  const datos = hoja.getDataRange().getValues();
  
  for (let i = 1; i < datos.length; i++) {
    const nombre = datos[i][0];
    const email = datos[i][1];
    const producto = datos[i][2];
    const total = datos[i][3];
    const estado = datos[i][4];
    const enviado = datos[i][5];
    
    if (estado === "Confirmado" && enviado !== "Enviado") {
      const asunto = `✅ Pedido confirmado — ${nombre}`;
      const cuerpo = `Hola ${nombre},\n\n`
        + `Tu pedido de "${producto}" por Q${total} está confirmado.\n`
        + `Tiempo de entrega: 3-5 días hábiles.\n\n`
        + `¡Gracias por tu compra!\nMente en Automático`;
      
      GmailApp.sendEmail(email, asunto, cuerpo);
      hoja.getRange(i + 1, 6).setValue("Enviado");
    }
  }
}

La columna F actúa como "candado" — una vez marcada como "Enviado", el script nunca vuelve a mandar ese correo aunque lo ejecutes mil veces.

🔴 SCRIPT 2

Alerta de pagos vencidos

Tu Sheet de cobros tiene: Cliente (col A), Email (col B), Monto (col C), Fecha de vencimiento (col D), Estado (col E). El script manda recordatorio a quienes llevan 7+ días sin pagar.

Apps Script — Alerta de pagos
function alertarPagosVencidos() {
  const hoja = SpreadsheetApp.getActiveSpreadsheet()
    .getSheetByName("Cobros");
  const datos = hoja.getDataRange().getValues();
  const hoy = new Date();
  let recordatoriosEnviados = 0;
  
  for (let i = 1; i < datos.length; i++) {
    const cliente = datos[i][0];
    const email = datos[i][1];
    const monto = datos[i][2];
    const vencimiento = new Date(datos[i][3]);
    const estado = datos[i][4];
    
    const diasVencido = Math.floor(
      (hoy - vencimiento) / (1000 * 60 * 60 * 24)
    );
    
    if (estado === "Pendiente" && diasVencido >= 7) {
      const asunto = `⚠️ Pago vencido — ${cliente}`;
      const cuerpo = `Estimado/a ${cliente},\n\n`
        + `Tu pago de Q${monto} venció hace ${diasVencido} días.\n`
        + `Por favor regulariza tu situación a la brevedad.\n\n`
        + `Escríbenos si tienes alguna duda.\nMente en Automático`;
      
      GmailApp.sendEmail(email, asunto, cuerpo);
      recordatoriosEnviados++;
    }
  }
  
  console.log(`Recordatorios enviados: ${recordatoriosEnviados}`);
}
⚡

Personaliza fácil con Claude: "Ajusta el script para que envíe el primer recordatorio a los 3 días, el segundo a los 7, y marque la fila como 'Moroso' a los 15." Claude lo adapta en segundos.

📊 SCRIPT 3

Reporte semanal automático

Cada lunes a las 8am, este script resume las ventas de la semana y se lo manda a tu equipo. Nadie tiene que hacer nada.

Apps Script — Reporte semanal
function reporteSemanal() {
  const hoja = SpreadsheetApp.getActiveSpreadsheet()
    .getSheetByName("Ventas");
  const datos = hoja.getDataRange().getValues();
  
  const hoy = new Date();
  const hace7dias = new Date(hoy - 7 * 24 * 60 * 60 * 1000);
  
  let total = 0, cantVentas = 0;
  let detalles = "";
  
  for (let i = 1; i < datos.length; i++) {
    const fecha = new Date(datos[i][0]);
    const cliente = datos[i][1];
    const monto = datos[i][2];
    
    if (fecha >= hace7dias) {
      total += monto;
      cantVentas++;
      detalles += `- ${cliente}: Q${monto}\n`;
    }
  }
  
  const cuerpo = `📊 REPORTE SEMANAL\n`
    + `=================\n`
    + `Ventas totales: ${cantVentas}\n`
    + `Monto total: Q${total}\n\n`
    + `DETALLE:\n${detalles}\n`
    + `Generado automáticamente por Apps Script`;
  
  GmailApp.sendEmail(
    "tu@email.com",
    "📊 Reporte semanal de ventas",
    cuerpo
  );
}
⏰ DISPARADORES

El truco: que funcione solo

Un script que tienes que ejecutar manualmente no es automatización. Los disparadores (triggers) son la magia que lo hace correr solo.

1

Abre el editor de Apps Script

Extensiones → Apps Script → pega tu código y guarda (Ctrl+S).

2

Ve a Disparadores (reloj)

En el panel izquierdo, clic en el ícono de reloj ⏰ o "Triggers" en la barra lateral.

3

Añade un nuevo disparador

Clic en "+ Añadir disparador" (esquina inferior derecha). Selecciona tu función.

4

Configura cuándo ejecutar

Elige: Basado en tiempo → Día → Lunes a las 8-9am. O cada hora, cada día — lo que necesites.

5

¡Listo! Funciona solo

Google ejecutará el script automáticamente. Puedes cerrar la computadora — no necesitas estar presente.

🔒

Primer uso: Google te pedirá autorización para que Apps Script use Gmail en tu nombre. Es normal — acepta los permisos. Solo se pide una vez.
