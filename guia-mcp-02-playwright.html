🎭 Sección 1

¿Qué es Playwright MCP y cómo funciona?

Playwright MCP convierte a Claude Code en un agente que puede abrir páginas web, hacer clic en botones, rellenar formularios, extraer información y tomar capturas de pantalla — todo de forma autónoma.

La diferencia técnica clave: en lugar de usar capturas de pantalla (como hacen otros agentes de IA), Playwright opera sobre el árbol de accesibilidad del navegador — una representación estructurada de todos los elementos de la página. Esto lo hace más preciso, más rápido y compatible con cualquier modelo de IA sin necesidad de visión artificial.

🎭

Analogía: Si el HTML de una web es como los cimientos de un edificio, Playwright lee el plano del edificio completo — cada puerta, cada botón, cada formulario — y puede interactuar con cualquier elemento de forma directa y precisa.

Navegadores compatibles

🧭

🌐

⚡ Sección 2

Instalación paso a paso

Playwright MCP no requiere API key — solo Node.js 18+. La instalación completa toma menos de 2 minutos.

1

Verifica Node.js

Abre tu terminal y escribe node --version. Necesitas v18 o superior. Si no lo tienes, descárgalo gratis en nodejs.org.

2

Instala con el comando oficial

💻 Terminal
claude mcp add playwright npx @playwright/mcp@latest
3

Reinicia Claude Code

Cierra completamente Claude Code y vuelve a abrirlo. El MCP se registra automáticamente al reiniciar.

4

Prueba que funciona

Escríbele a Claude: "Abre google.com en el navegador y dime el título de la página". Si responde con el título correcto, está funcionando.

💡

Pro tip: Para ejecutar sin ventana visible (ideal en servidores o automatizaciones en background), puedes configurar el modo headless. Agrega --headless como argumento en la configuración JSON del MCP.

🚀 Sección 3

5 casos de uso reales

Estos son los usos más valiosos de Playwright MCP en un flujo de trabajo real — sin necesidad de saber programar automatizaciones.

1

Scraping de webs dinámicas (SPAs)

Muchos sitios cargan el contenido con JavaScript después del HTML inicial. Playwright puede esperar a que cargue y extraer la información correcta. Ideal para webs de e-commerce, noticias o plataformas con login.

2

Testing automático de aplicaciones web

Claude puede recorrer tu web automáticamente: hacer clic en todos los botones, probar formularios, verificar que no haya errores. Un QA tester virtual que trabaja 24/7.

3

Rellenar formularios y automatizar flujos

¿Tienes que rellenar el mismo formulario muchas veces? Claude con Playwright puede hacerlo automáticamente. Registros, pedidos, actualizaciones de datos en portales — todo en piloto automático.

4

Monitorización de cambios en páginas

Visita una página de forma recurrente y alerta cuando cambia algo: precio de un producto, disponibilidad de tickets, actualizaciones en documentación técnica, cambios en webs de competidores.

5

Captura y análisis visual de webs

Toma capturas de pantalla automáticas de cualquier página para documentación, comparativas de diseño o para generar reportes visuales de cómo se ve tu producto en distintos navegadores.

💬 Sección 4

Ejemplos de prompts para Claude

Una vez instalado Playwright MCP, puedes pedirle a Claude cosas como estas directamente en el chat:

🔍 Scraping básico
"Ve a [URL] y extráeme todos los títulos y precios de los productos que aparecen en la página principal."
📋 Formularios automáticos
"Entra a [URL del formulario] y rellénalo con estos datos: [nombre, email, mensaje]. Luego dime si se envió correctamente."
📸 Captura de pantalla
"Toma una captura de pantalla de [URL] en desktop (1920px) y en móvil (390px) para comparar el diseño responsive."
🔄 Monitoreo de precios
"Ve a [URL del producto] y dime el precio actual. Necesito saber si cambió respecto a este precio de referencia: $49.99"
🧪 Testing de funcionalidad
"Prueba el formulario de contacto de mi web [URL]: rellena todos los campos con datos de prueba, envíalo y dime si aparece el mensaje de confirmación."
